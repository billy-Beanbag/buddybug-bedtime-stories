# Buddybug Docs

This folder holds planning and reference material that should stay separate from production app code.

## Design

- `docs/design/README.md` explains the current design artifacts and where to find them.
- `docs/website-structure.mmd` is the editable route map source.
- `docs/website-structure.html` is the rendered route map for quick viewing in a browser.

## Notes

- The standalone UI concept board now lives at `docs/design/ui-concepts.html`.
- Design prototypes should stay under `docs/design/` so they do not get mixed up with production app files.
