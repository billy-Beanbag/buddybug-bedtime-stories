# Design Artifacts

This area groups the non-production design references used while shaping Buddybug's UX and information architecture.

## Current files

- `../website-structure.mmd`
  Editable Mermaid source for the overall route and page map.

- `../website-structure.html`
  Browser-friendly rendered view of the Mermaid site structure.

- `ui-concepts.html`
  Standalone concept board showing visual experiments for the app shell, library, and reader.

## Working rule

Treat these files as references and prototypes, not as app runtime code. When a concept is approved, implement it in the real frontend and keep the prototype as supporting documentation only.
